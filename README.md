# BaseDev-SistemaAlunos
💻 Sistema de Gerenciamento de Alunos (Java)

Projeto desenvolvido com o objetivo de exercitar os fundamentos da programação em Java, abordando:

Algoritmos
Variáveis e tipos de dados
Estruturas condicionais (if/else)
Laços de repetição (for, while)
Estruturas de dados (Array, ArrayList)
Funções e métodos

🎯 Objetivo: Consolidar a base da lógica de programação através da construção de um sistema simples de gerenciamento de alunos via terminal.
    Projeto desenvolvido para fins de estudo e prática em programação.

▶️ Como executar o projeto:

Para executar o projeto no bloco de notas:
- Abra o Bloco de Notas;
- Copie e Cole o código no bloco de notas;
- Siga os passos, nomeie o arquivo com seguinte nome: "SistemaAlunos.java" e selecione o tiopo de arquivo para "Todos os arquivos", se não, ele será salvo como .txt e não irá funcionar. Salve o arquivo em sua área de trabalho.
- Vá em iniciar e abra o "Prompt de Comando" e digite: cd Desktop
- Digite o seguinte comando no prompt: javac SistemaAlunos.java
- Para executar o arquivo digite: java SistemaAlunos
- Seguindo esse passo a passo, irá aparecer o menu do sistema.

  Em caso de erro na execução:
  - Verifique a instalação java em seu dispositvo.
  - Verifique a escrita e o espaçamento, ela deve ser exata.
